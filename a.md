公司概况

一博科技成立于2003年，总部位于深圳，是一家专注于高速PCB设计和PCBA制造服务的技术型企业，2022年9月在深交所创业板上市，股票代码为301366。公司最早由多位具有华为背景的工程师创立，在高速PCB设计领域深耕二十余年。

官方主页：

一博科技官网

一、公司到底是做什么的？

简单来说，一博科技不是传统意义上的PCB板厂，而是一个面向电子硬件研发企业的一站式硬件创新平台。

其核心业务可以概括为：

帮客户把芯片方案变成真正能工作的硬件产品。

业务主要分为四大板块：

1. 高速PCB设计（核心竞争力）

这是公司最核心、壁垒最高的业务。

主要服务包括：

高速PCB Layout设计
SI（信号完整性）仿真
PI（电源完整性）仿真
EMC电磁兼容设计
高速接口设计

涉及的高速标准：

DDR4/DDR5
PCIe 4.0/5.0/6.0
100G/200G/400G网络
112G/224G SerDes
HBM相关接口

公司拥有900余名PCB设计工程师，是国内规模最大的高速PCB设计团队之一。

2. PCB制造

公司在珠海拥有全资PCB工厂：

珠海市一博电子有限公司

主要生产：

高多层PCB
高速板
HDI板
埋盲孔板
背钻板

技术能力：

项目	能力
最大层数	132层
产品类型	高速、高多层、高阶HDI
交付速度	20层以内最快8天

主要应用：

AI服务器
光模块
通信设备
自动测试设备（ATE）
工控设备
医疗设备

3. PCBA制造

PCBA即：

PCB + 元器件 + SMT贴片 + 焊接 + 测试

公司在深圳、上海、成都、长沙、天津、苏州等地设有工厂，总计拥有60余条SMT产线。

主要提供：

SMT贴片
DIP插件
整机组装
功能测试
老化测试

定位：

研发打样 + 中小批量生产

这是区别于很多大型EMS厂的重要特点。

4. 元器件选型与供应链服务

公司拥有大量电子元器件库存，可以帮助客户：

BOM优化
元器件选型
替代料推荐
缺料解决方案

对于芯片公司和初创企业而言：

可以显著降低研发周期。

二、公司的商业模式

公司的商业模式非常类似：

“电子行业的研发外包平台 + 小批量制造平台”

举个例子：

假设一家AI芯片公司设计出了一颗新芯片。

接下来需要：

设计测试板
进行高速仿真
制作PCB
焊接样机
调试测试

这些工作很多芯片公司自己并不做。

于是就交给一博科技。

因此：

芯片公司负责芯片设计，一博负责把芯片变成可运行的硬件系统。

公司客户以高端研发客户为主，每年服务客户超过3000家。

四、为什么市场关注一博科技？
1. AI算力产业链受益者

AI服务器需要：

GPU
CPU
光模块
高速交换芯片

而这些设备都需要：

高速PCB设计 + 高速背板 + 信号完整性仿真

AI服务器速度越高：

对PCB设计能力要求越高。

因此，一博是典型的：

AI硬件基础设施“卖铲人”。

2. 国内高速PCB设计龙头

国内真正具备：

PCIe 5/6
DDR5
112G/224G
SI/PI仿真

完整能力的企业并不多。

公司在这一细分领域具备较强技术壁垒。

3. 国产芯片研发受益

随着国产算力芯片、自研服务器的发展：

研发测试板需求增加。

公司长期服务于芯片验证环节，因此会间接受益于国产替代趋势。

五、公司的优势与风险
优势
（1）技术壁垒高

高速PCB设计经验积累需要多年工程实践。

（2）客户粘性强

客户一旦验证通过，通常不会轻易更换供应商。

（3）轻资产属性

设计服务业务毛利率较高。

（4）覆盖研发全流程

设计→制板→贴片→测试一体化。

风险
（1）下游研发投入波动

半导体和通信行业景气度下降时，研发订单可能减少。

（2）客户集中于高科技行业

受半导体周期影响较明显。

（3）制造业务竞争激烈

PCB和SMT行业整体竞争仍较充分。

六、如果用一句话总结

一博科技是一家以高速PCB设计为核心、向芯片公司和高端电子企业提供“从设计到样机制造”一站式服务的硬件研发平台，是国内高速PCB设计领域的龙头企业，也是AI服务器和国产芯片产业链中的重要“卖铲人”。

如果你把一博科技放在整个PCB产业链中去看，它确实是一家很有特色的公司，但也存在一些比较明显的短板和成长瓶颈。

我觉得可以从商业模式、行业属性、竞争格局、成长空间四个角度来看。

一、一博科技最大的缺点：规模天花板相对有限

这是市场一直给予其估值折价的重要原因。

一博的核心业务是：

高速PCB设计 + 研发打样 + 中小批量PCBA

这种模式决定了：

技术含量高；
毛利率较高；
客户粘性强；

但同时：

很难像制造业那样快速复制和扩张。

因为高速PCB设计本质上属于：

「工程师密集型行业」。

收入增长很大程度依赖：

工程师人数 × 人均产值

例如：

新增100名工程师；
培养成熟可能需要2~3年；
高级高速PCB设计人才稀缺。

因此公司的扩张速度天然受到人才供给约束。

二、对核心人才依赖非常大

一博并不是一个标准制造企业。

它更像：

「硬件研发服务公司」。

公司最重要资产不是厂房，而是：

高速PCB设计专家；
SI/PI仿真工程师；
EMC工程师；
资深Layout工程师。

问题在于：

这些人才培养周期极长。

一个优秀的高速PCB工程师：

通常需要：

5~10年以上项目经验。

如果未来：

行业竞争加剧；
核心员工流失；
薪资持续上涨；

都会影响公司竞争力和利润率。

三、业务偏研发阶段，容易受行业景气影响

一博大量客户来自：

半导体公司；
通信设备公司；
AI服务器公司；
工业控制企业。

这些行业有一个共同特点：

研发投入具有明显周期性。

经济不好时：

企业通常最先削减的支出之一就是：

新产品研发预算。

所以：

当半导体行业进入下行周期时：

一博订单可能受到较大影响。

相比之下，像：

深南电路
沪电股份

这种大量进入稳定量产阶段的企业，抗周期能力往往更强。

四、制造规模远不如行业龙头

虽然一博也做：

PCB制板；
PCBA制造；

但其制造规模和行业龙头相比差距较大。

例如：

公司	核心优势
一博科技	设计服务
深南电路	高端制造
沪电股份	AI服务器PCB量产
胜宏科技	AI高阶PCB量产
鹏鼎控股	全球规模制造

因此：

如果未来AI硬件进入全面放量阶段：

大量利润可能仍然由大型制造厂商获取。

一博更多赚的是：

研发验证阶段的钱。

这意味着：

成长弹性可能不如纯AI PCB制造龙头。

五、行业竞争正在加剧

过去高速PCB设计市场竞争者较少。

但近年来：

金百泽
兴森科技

都在加强：

高速设计能力；
SI/PI仿真能力；
一站式研发服务能力。

如果未来竞争进一步加剧，而公司无法持续保持技术领先：

可能出现：

客户流失；
毛利率下降；
市场份额被侵蚀。

这一风险在公司招股书中也有明确提示。

六、未来发展过程中可能遇到的几个关键阻碍
1. AI景气度下降

目前市场给予一博较高关注，核心逻辑之一就是：

AI服务器、光模块、算力卡需求增长。

如果未来AI投资降温：

相关研发需求可能下降。

公司业绩增速也可能放缓。

2. 从设计服务向制造扩张存在挑战

公司正在向：

高速高阶PCB制造；
光模块PCBA；
ATE测试板；

等领域延伸。

但制造业与设计服务完全不同。

制造业竞争的是：

资本投入；
良率控制；
规模效应；
成本控制。

能否顺利完成从“设计驱动”到“设计+制造”的升级，还有待持续验证。

3. AI时代可能出现新的技术替代

随着EDA自动化和生成式AI的发展，部分传统PCB设计流程未来可能被自动化工具替代。

虽然短期内高端高速设计仍高度依赖资深工程师，但长期来看：

工程师效率提升，也可能压缩纯设计服务公司的价值空间。

总结

我认为，一博科技最核心的问题可以概括成一句话：

它是一家优秀的小而美技术服务公司，但要成长为类似深南电路那样的大市值平台型企业，仍面临人才瓶颈、规模瓶颈和制造扩张瓶颈。

从投资角度看，一博更像：

“国产芯片和AI研发投入的卖铲人”。

如果未来中国半导体研发持续高景气，它会持续受益；但如果行业进入资本开支下行周期，其业绩波动可能会比市场预期更大。
